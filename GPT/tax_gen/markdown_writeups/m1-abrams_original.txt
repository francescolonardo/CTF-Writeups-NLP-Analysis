<h2>Solution</h2>

<p>Connecting to the URL you will find a default Apache2 installation page. The version of the server is <code>Apache/2.4.29 (Ubuntu)</code>.</p>

<p>Performing an enumeration will let you to discover the following.</p>

<p>```
user@machine:~$ sudo dirb http://challenges.auctf.com:30024/ /usr/share/dirb/wordlists/vulns/apache.txt -x /usr/share/dirb/wordlists/extensions_common.txt -w</p>
