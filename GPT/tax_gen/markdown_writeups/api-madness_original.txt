<h2>Solution</h2>

<p>Connecting to <code>http://challenges.auctf.com:30023</code> you will get the following message.</p>

<p><code>
{
  "NOTE": "For API help visit our help page /static/help", 
  "status": "OK"
}
</code></p>

<p>The help page at <code>http://challenges.auctf.com:30023/static/help</code> will give you the following information.</p>

<p>```html
<html>
<body>
<center><h1>FTP Server API Help Page</h1></center>
<br>
<br></p>

<h2>Endpoints</h2>

<p>/api/login - POST</p>

<p>/api/ftp/dir - POST</p>

<p>/api/ftp/get_file - POST</p>

<p></br>
</br></p>

<h2>Params</h2>

<p>/api/login - username, password</p>

<p>/ftp/dir - dir</p>

<p>/ftp/get_file - file</p>

<p></body>
</html>
```</p>

<p>So the following API endpoints are available:
* <code>/api/login</code> - <code>HTTP POST</code> with <code>username</code>, <code>password</code> JSON parameters;
* <code>/api/ftp/dir</code> - <code>HTTP POST</code> with <code>dir</code> JSON parameter;
* <code>/api/ftp/get_file</code> - <code>HTTP POST</code> with <code>file</code> JSON parameter.</p>

<p>Using the <code>/api/login</code> endpoint an error page will appear.</p>
