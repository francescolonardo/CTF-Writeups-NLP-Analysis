Title: Bypassing Client-Side Validation for Arbitrary File Upload
Primary topic: Web Exploitation
Secondary topic: File Uploads
Tools used: Burp Suite
Keywords: client-side validation, remote code execution, PHP file, flag file

#overview
During a CTF event, I found a web application that allowed users to upload files. However, the application appeared to have client-side validation in place, restricting the file types that could be uploaded.

#analysis
This caught my attention, as bypassing client-side validation could allow me to upload arbitrary files, potentially leading to remote code execution or other security issues. To bypass the client-side validation, I intercepted the HTTP request using a proxy tool like Burp Suite and modified the file type to an unsupported one, such as a PHP file. For example, if the original request had a parameter like Content-Type: image/jpeg, I modified it to Content-Type: application/x-php. Once the modified request was sent, the web application accepted the unsupported file type, confirming that the client-side validation could be bypassed.

#attack_execution
To exploit the vulnerability and obtain the flag for the Web Exploitation challenge, I crafted a PHP file that contained code to read the content of the flag file. For example, the PHP file could contain the following code:

```
<?php
  $content = file_get_contents('/path/to/flag.txt');
  echo $content;
?>
```

After uploading the crafted PHP file, I navigated to the location where the uploaded files were stored, such as http://example.com/uploads/my-php-file.php. The web application executed the PHP code, displaying the content of the flag file and allowing me to obtain the flag needed to complete the challenge.

This CTF writeup highlights the importance of implementing server-side validation for file uploads in web applications to prevent security vulnerabilities. By bypassing the client-side validation, I was able to upload an arbitrary file, leading to the exposure of sensitive data and obtaining the flag needed to complete the challenge.
