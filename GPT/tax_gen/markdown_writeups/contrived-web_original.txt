<h1>Contrived-Web</h1>

<h3>TL;DR</h3>

<ul>
<li>SSRF in the /api/image to ftp server</li>
<li>CRLF injection in the username to inject ftp commands</li>
<li>Upload file into the ftp server using profile picture upload</li>
<li>Use PORT ftp command to SSRF to rabbitmq http API (use REST to discard the png header from the uploaded file)</li>
<li>Inject in rabbitmq email queue with an "attachment" parameter to get it to email us the flag</li>
</ul>
