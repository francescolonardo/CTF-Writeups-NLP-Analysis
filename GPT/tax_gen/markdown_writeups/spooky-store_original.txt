<h2>Solution</h2>

<p>The web page sends data via XML envelops, using two Javascript files, when buttons are pressed.</p>

<p>The first script (<code>http://web1.utctf.live:5005/static/js/xmlLocationCheckPayload.js</code>) crafts the XML envelope.</p>

<p>```javascript
window.contentType = 'application/xml';</p>

<p>function payload(data) {
    var xml = '<?xml version="1.0" encoding="UTF-8"?>';
    xml += '<locationCheck>';</p>

<pre><code>for(var pair of data.entries()) {
    var key = pair[0];
    var value = pair[1];

    xml += '&lt;' + key + '&gt;' + value + '&lt;/' + key + '&gt;';
}

xml += '&lt;/locationCheck&gt;';
return xml;
</code></pre>

<p>}
```</p>

<p>The second script (<code>http://web1.utctf.live:5005/static/js/locationCheck.js</code>) sends the XML envelope and reads the answer.</p>

<p>```javascript
document.querySelectorAll('.locationForm').forEach(item =&gt; {
    item.addEventListener("submit", function(e) {
        checkLocation(this.getAttribute("method"), this.getAttribute("action"), new FormData(this));
        e.preventDefault();
    });
});
function checkLocation(method, path, data) {
    const retry = (tries) =&gt; tries == 0
        ? null
        : fetch(
            path,
            {
                method,
                headers: { 'Content-Type': window.contentType },
                body: payload(data)
            }
          )
            .then(res =&gt; res.status == 200
                ? res.text().then(t =&gt; t)
                : "Could not fetch nearest location :("
            )
            .then(res =&gt; document.getElementById("locationResult").innerHTML = res)
            .catch(e =&gt; retry(tries - 1));</p>

<pre><code>retry(3);
</code></pre>

<p>}
```</p>
