<h2>Solution</h2>

<p>This challenge gives provides you a website with the ability to upload a barcode. The website will "scan" this barcode and will return information about the movie title that was encoded.</p>

<ol>
<li>To solve this challenge, generate a barcode with a SQL injection that returns all rows using an online barcode generator.
<ul>
<li>For example, <code>' OR 1=1;  --</code></li>
</ul></li>
<li>Upload the barcode to the website.</li>
<li>The flag can be found at the bottom of the returned results.</li>
</ol>
