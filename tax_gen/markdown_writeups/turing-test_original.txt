<h2>Solution</h2>

<p>The shown web page seems to perform a password reset functionality via security questions. It seems to be related to the account recovery of <em>Alan Turing</em>.</p>

<p>If you submit data, the web page will highlight in red wrong fields.</p>

<p>Some recon activities on the Internet about Alan Turing must be done in order to answer security questions.</p>

<p>After few attepts you will discover all the answers:
* Text field - <em>Mother's Maiden Name</em> - Stoney
* Text field - <em>First School Attended</em> - St. Michael's
* Text field - <em>Favorite Primary School Subject</em> - Science
* Text field - <em>Favorite Olympic Event</em> - Marathon
* Text field - <em>2 + 2 - 3 = ?</em> - 1
* Checkbox - <em>Is it a leap year?</em> - checked
* Checkbox - <em>I agree Security Questions are Bad.</em> - checked</p>
