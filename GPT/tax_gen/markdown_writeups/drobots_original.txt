<h2>Drobots</h2>

<blockquote>
  <p>Solved by Legend</p>
</blockquote>

<p>Challenge description </p>

<blockquote>
  <p>Pandora's latest mission as part of her reconnaissance training is to infiltrate the Drobots firm that was suspected of engaging in illegal activities. Can you help pandora with this task?</p>
</blockquote>

<p>In this challenge the website is showing a login page.</p>

<p><img src="https://i.imgur.com/uxHR1gT.png" alt="" /></p>

<p>Since no other info was there or in the source code of the page I checked the docker config file.</p>

<p>The website is a <code>Flask</code> application running on <code>MySQL</code> so I thought of SQL Injection cand be done. Here I found interesting thing that in <code>config.py</code> file they have provided the database information.</p>

<p><img src="https://i.imgur.com/07ijody.png" alt="" /></p>

<p>Also in the <code>database.py</code> also they have given the hint that the input is not sanitized. 
The logic of the database is that we need the <code>token</code> of the password to login. So if we can find the token we can login.</p>

<p><img src="https://i.imgur.com/hO6kBee.png" alt="" /></p>

<p>Then I saved the login request and gave it to <code>sqlmap</code> though which I was able to extract the details.</p>

<pre><code>sqlmap -r ./req --dbms=mysql -D drobots -T users --dump
