<h1>Hungry Man - Web</h1>

<p>Author - <a href="https://github.com/UnknownAbyss">UnknownAbyss</a></p>

<p>Requirements : Python, cURL</p>

<hr />

<p>The challenge gives us the following text -</p>

<blockquote>
  <p>There is nothing here I promise! ;)</p>
</blockquote>

<p><code>
very.uniquename.xyz:2095
</code></p>

<h4>Upon opening the link we see -</h4>

<p align="center"><img src="https://i.imgur.com/HsXPiDO.png" alt="Chall Image"></p>

<p>Checking the network tab, we find the following response header</p>

<p align="center"><img src="https://i.imgur.com/sYB2iGY.png" alt="Chall Image"></p>

<p>We send the following curl request setting the cookie to the given string</p>

<p><code>sh
curl -H "Cookie: flag=c29tZXRpbWVzIHRoZSBrZXkgdG8gdW5sb2NraW5nIHRoZSBhbnN3ZXIgaXMgdGhlIHF1ZXN0aW9uIGl0c2VsZi4uLiBidXQgbXkgZmF2b3JpdGUgaXMgY2hvY28gY2hpcA==" very.uniquename.xyz:2095
</code></p>

<p>The repsonse is...</p>

<p><code>
{"flag":"522748524ad010358705b6852b81be4c"}
</code></p>

<p>Then sending this new cookie...</p>

<p><code>sh
curl -H "Cookie: flag=522748524ad010358705b6852b81be4c" very.uniquename.xyz:2095
</code></p>

<p>The repsonse is...</p>

<p><code>
{"flag":"70e9490b5d5a217070c1e7df9518e9d5"}
</code></p>

<p>Let us write a script for this...</p>

<p>```py
import requests</p>

<p>url = 'http://very.uniquename.xyz:2095'
flags = ['c29tZXRpbWVzIHRoZSBrZXkgdG8gdW5sb2NraW5nIHRoZSBhbnN3ZXIgaXMgdGhlIHF1ZXN0aW9uIGl0c2VsZi4uLiBidXQgbXkgZmF2b3JpdGUgaXMgY2hvY28gY2hpcA==']</p>

<p>while (flags[-1]!='EOF'):
    x = requests.get(url, cookies={'flag': flags[-1]}).text
    flags.append(x[9:-3 ])</p>

<p>for i in flags[1:-1]:
    print(i)
```</p>
