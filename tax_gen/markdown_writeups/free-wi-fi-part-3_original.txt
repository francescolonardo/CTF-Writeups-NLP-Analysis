<p>You have discovered that:
1. a <code>/forgotpassword.html</code> page exists;
2. <code>true.grit@umbccd.io</code> is a user of the system;
3. the forgot password functionality uses two different fields for username and e-mail, with a JavaScript code to copy the value inserted into the input field.</p>

<p>As a consequence, it is sufficient to intercept the request and change the e-mail with one you control, leaving the discovered username.</p>
