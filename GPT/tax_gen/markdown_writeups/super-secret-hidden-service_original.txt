<h2>Solution</h2>

<p>Connecting to the website the following message is prompted.</p>

<p><code>
421 Site 138.247.13.115 is not served on this interface
</code></p>

<p>The website is under HTTPS, but the certificate seems invalid. Analyzing the certificate, you will discover that it was created for the host <code>138.247.13.115.xip.io</code>.</p>

<p>Connecting to <code>https://138.247.13.115.xip.io/</code> will print the flag.</p>
