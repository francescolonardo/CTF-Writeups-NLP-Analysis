<ul>
<li>User cookie is double base64 encoded, so we double decoded and ended up with the following :</li>
</ul>

<p><code>
O:4:"user":3:{
    s:8:"username";s:4:"user";
    s:5:"admin";b:0;
    s:11:"profile_img";s:23:"/var/www/html/guest.png";
}
</code></p>

<p><code>
O   -&gt; Object
O:4 -&gt; Object of length 4
:3: -&gt; Object has 3 attributes
s   -&gt; String data type
s:8 -&gt; String of length 8
b   -&gt; Boolean data type
b:0 -&gt; False
</code></p>

<ul>
<li>This was a <strong>PHP Object Injection</strong> challenge</li>
<li>The goal of this challenge was not to become admin and if we try then the server sends hints or funny responses</li>
<li>Directly accessing the flag did not work so we can simply use <code>../</code></li>
<li>Final payload :</li>
</ul>
