<h2>Quick</h2>

<p>Python script to calculate and send the SHA-256 hash of a string.</p>

<p>```python
import requests
import re
import hashlib</p>

<p>s = requests.Session()
content = s.get("http://18.139.27.125/UMJVHRV5/")
match = re.search(r"[A-Fa-f0-9]{32}", content.content.decode())</p>

<p>hash<em>object = hashlib.sha256(match.group(0).encode())
hex</em>dig = hash<em>object.hexdigest()
myobj = {'hash': hex</em>dig, 'execute': ''}
x = s.post("http://18.139.27.125/UMJVHRV5/", data=myobj)
print(x.text)
```</p>
