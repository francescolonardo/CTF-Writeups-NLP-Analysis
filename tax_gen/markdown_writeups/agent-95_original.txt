<h1>Exploitation</h1>

<p>The challenge is very simple once you know what to do. </p>

<p><code>
We will only give our flag to our Agent 95! He is still running an old version of Windows...
</code></p>

<p>We assume Agent 95 uses Windows 95. So just change the <code>User-Agent</code> header to <code>Mozilla/4.0 (compatible; MSIE 4.01; Windows 95)</code>.</p>

<p>```python</p>

<blockquote>
  <blockquote>
    <blockquote>
      <p>import requests
      r = requests.get('http://jh2i.com:50000', headers={'User-Agent': 'Mozilla/4.0 (compatible; MSIE 4.01; Windows 95)'})
      r.text
      'flag{user<em>agents</em>undercover}\n<div style="text-align:center">\n<br><br><br><br>\n<b> NOT CHALLENGE RELATED:</b><br>THANK YOU to Digital Ocean for supporting NahamCon and NahamCon CTF!\n<p>\n<img width=600px src="https://d24wuq6o951i2g.cloudfront.net/img/events/id/457/457748121/assets/1b5a9739fd31b42fa4eb37ac6b3a6e1c.DOlogo.png">\n</p>\n</div>'
      ```</p>
    </blockquote>
  </blockquote>
</blockquote>

<p>There, you got the flag.</p>

<p><code>
flag{user_agents_undercover}
</code></p>
